# Car Price Prediction

A Machine Learning web app that predicts the selling price of a car based on various features.

## Live Demo
(https://car-price-prediction-04.streamlit.app/)

##  Features
- Predict car price instantly
- User-friendly interface using Streamlit
- Based on real dataset
- Accurate predictions using Random Forest

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Model
- Random Forest Regressor
- Feature engineering (Car Age)
- One-hot encoding for categorical variables
